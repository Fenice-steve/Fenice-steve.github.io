简介

Git是一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。当初Linus Torvalds(Linux操作系统的作者)是为了帮助管理Linux内核开发而开发的一个开放源码的版本控制软件。而现在它显然以成为最先进的分布式版本控制系统(没有之一)。
下面就是Git的time show了
